# 📱 TaskFlow – Gerenciador de Tarefas

## 📌 Sobre o projeto
O TaskFlow é um aplicativo Android desenvolvido com o objetivo de auxiliar na organização de tarefas do dia a dia. O app permite criar, visualizar, marcar como concluídas e excluir tarefas de forma simples e intuitiva.

Este projeto foi desenvolvido como prática de desenvolvimento mobile, aplicando conceitos fundamentais de programação, interface e manipulação de dados.

## 🚀 Funcionalidades
* ✅ Adicionar novas tarefas
* ✅ Listar tarefas em tempo real
* ✅ Marcar tarefas como concluídas
* ✅ Excluir tarefas
* ✅ Interface simples e fácil de usar

## 🛠️ Tecnologias utilizadas
* **Kotlin**
* **Android Studio**
* **Android SDK**
* **RecyclerView**
* **ViewBinding**
